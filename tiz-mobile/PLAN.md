# Tiz Mobile MVP

## 1️⃣ 底部 Dock

```
[ 消息 ]   [ 发现 ]   [ 我的 ]
```

| 按钮  | 核心定位    | 功能说明                                           |
| --- | ------- | ---------------------------------------------- |
| 消息  | 提醒与通知中心 | AI 或系统提醒、Webhook 事件推送                          |
| 发现  | 功能/内容探索 | 顶部多 Tab：翻译（核心功能，多语言支持）、测验（占位，显示"敬请期待"）         |
| 我的  | 账号与配置   | 账号信息、偏好设置（主题、语言、Expert Mode）、Webhook 配置、App 设置 |

---

## 2️⃣ 消息页

- 显示 AI 提醒或条件触发事件
- 支持 Webhook 推送消息
- 消息内容可包含标题、摘要、跳转链接

---

## 3️⃣ 发现页

### 翻译 Tab（核心 MVP）

- 输入文本 → AI 翻译
- 多语言支持（英语、粤语、川语）
- 选择源语言和目标语言
- 输出区域展示翻译结果，可复制/分享

### 测验 Tab（占位）

- 暂不实现题目功能
- 显示占位提示，保留位置便于未来扩展

---

## 4️⃣ 我的页

- **账号信息**：邮箱/昵称，登出/切换账号
- **偏好设置**：默认语言、主题、Expert Mode
- **Webhook 配置**：添加/编辑/删除 URL
- **App 设置**：清除缓存、查看版本信息

---

## 5️⃣ MVP 核心原则

1. **极简极客** → 主题就黑或者白就可以
2. **界面完整** → 留测验占位，保证未来扩展
3. **工具感强** → 快速访问翻译，消息提醒可用
4. **后续可扩展** → 支持测验、头脑风暴或其他 AI 功能
5. **Webhook 集成** → 外部系统可发送通知到 App，增强互动性
