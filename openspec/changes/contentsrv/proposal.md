# contentsrv

## Why

内容服务是核心业务服务，管理题库、题目、分类和标签。

## What Changes

### 1. 题库管理

- 题库列表 (分页、筛选、搜索)
- 题库详情
- 题库创建/更新/删除 (软删除)

### 2. 题目管理

- 题目生成 (调用 llmsrv)
- 题目列表
- 批量获取题目

### 3. 分类和标签

- 分类列表
- 标签列表

### 4. 内部 API

- 供其他服务获取题库和题目

## Scope

### In Scope

- 题库 CRUD
- 题目管理
- 分类和标签查询
- 内部 API

### Out of Scope

- AI 生成逻辑 (llmsrv 负责)
- 练习/测验逻辑 (各自服务负责)
